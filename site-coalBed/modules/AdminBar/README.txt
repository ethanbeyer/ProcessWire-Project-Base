AdminBar for ProcessWire 2.1 by @apeisa

// INSTALLATION
Just put all files into /site/modules/AdminBar/ folder, then go admin -> modules -> and install AdminBar

// PERMISSIONS
If you want to use AdminBar for other users than superuse, please add permission "adminbar" to their role. If you are upgrading from older version of AdminBar, you might need to uninstall/install to see that permission. Or you can add it by yourself.

// MINOR ISSUES
-Not yet tested IE8 & below
-Has annoying popup alert in IE9 if template contains TinyMCE-inputfields.

// BROWSER SUPPORT

-FF3+
-Webkit family (Chrome, Safari)
